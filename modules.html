composeComparators

  • composeComparators<T>(comparators: (undefined | null | Comparator<T>)[]): Comparator<T>
    • Defined in factories.ts:63

    Composes multiple comparators into a single new one. The new comparator will call them in order, if the current comparator defined the values as equal.

    Nullish values in the initial array of comparators will be removed, before creating the composed comparator.

    example

    const myStringComparator = composeComparators([byLength, localeCompare]);

Const naturalOrder

  • naturalOrder(a: any, b: any): number
    • Defined in comparators.ts:20

    Compares the values a and b by their natural order via the < operator. Returns 0 if the values are strictly equal.

    example
    // ascending
    [0, 3, 5, 1, 2, 4].sort(naturalOrder) == [0, 1, 2, 3, 4, 5];
